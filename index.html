Work

News API

A news API. This is available on Heroku and the repository is available here.

Roast Dinner

A small app that can be used to generate the timings for cooking a roast dinner. Simply enter the meat, its weight and the time you wish to eat. All the timings will be generated for you, allowing a stress-free roast experience.

This app was built in javascript with full TDD using mocha and chai. Bootstrap utilised for the front-end. Try it out.

News Tone Analyser

A simple app that analyses the current top articles from a selected media outlet, displaying the overarching tone of those articles. It is available here and the repository is available on Github.

Thanks in particular for the assistance of Colum Lynagh in completing this project.
